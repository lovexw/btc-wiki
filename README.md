# 比特币技术百科 - Bitcoin Wiki 中文版

一个全面、详细、通俗易懂的比特币技术知识库。

## 🚀 特点

- **详尽全面**：覆盖比特币的所有核心技术概念
- **通俗易懂**：用日常语言解释复杂概念，配合丰富的例子和类比
- **结构清晰**：内容组织有序，便于学习和查阅
- **中文内容**：完全中文讲解，消除语言障碍
- **代码示例**：提供实际的代码演示和说明
- **图文并茂**：丰富的示意图和流程图（使用文本表示）
- **左侧浮动导航**：方便随时切换不同主题学习，支持展开/收起
- **现代化设计**：采用简洁优雅的 HAB-B 风格标准

## 📚 内容主题

本Wiki涵盖以下核心主题：

1. **比特币基础** - 了解比特币的起源、定义和基本概念
2. **区块链技术** - 深入理解区块链的数据结构和工作原理
3. **密码学原理** - 哈希函数、公钥密码学、数字签名详解
4. **挖矿机制** - 工作量证明、难度调整、挖矿过程详解
5. **交易系统** - UTXO模型、交易结构、交易验证机制
6. **P2P网络** - 节点通信、区块传播、网络协议
7. **共识机制** - 最长链规则、分叉处理、51%攻击
8. **脚本系统** - Bitcoin Script语言、锁定脚本、解锁脚本
9. **钱包技术** - 地址生成、私钥管理、HD钱包
10. **安全性分析** - 常见攻击、安全最佳实践、漏洞防范
11. **高级主题** - SegWit、闪电网络、Taproot升级
12. **实现细节** - 源码解析、数据存储、性能优化

## 📖 使用方法

### 在线访问

直接在浏览器中打开 `index.html` 文件即可访问。

### 本地部署

1. 克隆或下载此仓库
2. 用浏览器打开 `index.html`
3. 使用左侧浮动导航或主题卡片浏览不同章节

### 导航功能

- **左侧浮动导航**：点击左上角的菜单图标可以展开/收起导航栏
- **快速切换**：在导航栏中点击任意主题即可快速跳转
- **当前主题高亮**：当前正在查看的主题会在导航栏中高亮显示
- **响应式设计**：在移动设备上也能完美使用

### 搜索功能

在首页搜索框输入关键词（如"挖矿"、"交易"、"密码"等）快速找到相关内容。
